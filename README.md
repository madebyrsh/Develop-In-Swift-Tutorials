## ✨ About

### 🇰🇷 Korean

이 저장소는 Apple의 **Develop in Swift Tutorials**를 기반으로  
Swift 앱 개발을 학습하고 기록하는 공간입니다.

단순한 코드 저장이 아니라,

- 직접 구현하고  
- 이해한 내용을 정리하고  
- 학습 과정에서의 생각과 회고를 남기는  

**개발 성장 로그**입니다.

---

### 🇺🇸 English

This repository is based on Apple’s **Develop in Swift Tutorials**,  
serving as a space to learn and document Swift app development.

Rather than simply storing code, this repository focuses on:

- Building features hands-on  
- Organizing learned concepts  
- Reflecting on the learning process  

This is a **development growth log**.
